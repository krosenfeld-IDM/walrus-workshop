# README

## Setup

Setup environment with uv: `uv sync` and make sure you have dependencies:

**FFmpeg**
```
sudo apt update
sudo apt install ffmpeg
```
**The well**
Download limited dataset for example:
```
the-well-download --dataset turbulent_radiative_layer_2D
```
